<h1>COMO CONFIGURAR EL PROYECTO Y JUGAR</h1>

Clonar o descargar el proyecto desde GitHub. Luego, compilar el "TP-LYB-Server" y ejecutar en la carpeta "bin" el ejecutable.
Luego, compilar y ejecutar el cliente "TP-Laberintos-y-Fantasmas". El cliente se conectará al servidor.
